## 如何参与

1. 点击右上角 Fork 按钮，将仓库复制到你的账号下
2. 克隆你 Fork 的仓库到本地
3. 创建新分支
4. 在下方「学习打卡」区域添加你的记录
5. 提交并推送到你的 Fork 仓库
6. 向本仓库发起 Pull Request
